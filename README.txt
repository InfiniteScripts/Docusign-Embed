Wordpress plugin to use with Docusign's API for embedding signable PDFs.

Adds a settings section to WP admin. You'll need a Docusign API and account login set here. You can upload any PDF in the settings section.

Then simply embed the form using [docusign_embed].

Please feel free to email me with any bugs or feature requests.
